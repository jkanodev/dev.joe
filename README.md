# Joey Kano — DevOps Portfolio Project

This repository documents a real-world DevOps workflow built from scratch,
covering containerization, CI/CD automation, image distribution, and live cloud deployment.

This is not a tutorial project.
This is a working system deployed on real infrastructure.

---

## What This Project Demonstrates

- Docker containerization using NGINX
- CI automation with GitHub Actions
- Public image distribution via Docker Hub
- Cloud deployment on AWS EC2
- Secure access using SSH key authentication
- Cost-aware infrastructure management

---

## Architecture Overview

Local Development
→ Docker Image
→ GitHub Repository
→ GitHub Actions CI
→ Docker Hub Registry
→ AWS EC2
→ Public Web Access

---

## Live Deployment

- Cloud: AWS EC2
- OS: Amazon Linux 2023
- Instance: t3.micro
- Runtime: Docker
- Port: 80 (HTTP)
